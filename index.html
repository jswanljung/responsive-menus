Exempel på responsiva menyer med enkel javascript och CSS

Källkoden finns här.

Testa att ändra bredden på skärmen! Bäst funkar det att använda utvecklingsverktygen i t.ex. Chrome.

Det här är ett exempel på hur javascript och CSS kan användas ihop för att implementera olika effekter, i det här fallet ihopvikbara menyer. Det är möjligt att ändra ett html-elements stilegenskaper direkt i javascript, men att göra på det sättet blir krångligt om sidan ska kunna anpassas till olika plattformer (dator, mobil, surfplatta, osv.). Ett lättare sätt är att definiera olika klasser i CSS för elementets olika tillstånd (meny öppen, meny stängd). Sen använder vi javascript för att lägga till eller ta bort klasser när användaren klickar på menyerna. Eftersom vi styr klasser och inte stilar direkt, kan vi använda identisk javascript både till breda skärmar där vi har en så kallad "rullgardinsmeny" och till smala mobilskärmar där menyn är vertikalt staplad.

Både skript och css är ganska väldokumenterade. Menyerna hanterar "tabbning" genom länkar ganska bra men ett komplett menysystem borde också kunna navigeras med pilar. Utrymme för förbättring finns! CSS-transitioner kunde också användas för att få menyerna att kännas lite mer dynamiska. Varsågoda att låna och experimentera!
